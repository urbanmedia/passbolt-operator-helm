A Helm chart for setting up a passbolt secret in Kubernetes

## Installation

### Add Helm repository

```bash
helm repo add passbolt-operator https://urbanmedia.github.io/passbolt-operator-helm
```

### Add a secret

```bash
helm install \
  my-secret passbolt-operator/passbolt-secret \
  -f values.yaml
```
